### Definitions

**Half Range Expansions**
A half range expansion is the representation of a function $f(x)$ defined on an interval $(0, L)$ in terms of a series of trigonometric functions.
*   **Sine Series:** Generated by eigenfunctions satisfying boundary conditions $y(0) = 0$ and $y(L) = 0$.
*   **Cosine Series:** Generated by eigenfunctions satisfying boundary conditions $y'(0) = 0$ and $y'(L) = 0$.

**Eigenvalues and Eigenfunctions**
In the context of the differential equation $y'' + \lambda y = 0$, the values of $\lambda$ for which nontrivial solutions exist are called **eigenvalues**, and the corresponding solutions are **eigenfunctions**. These eigenfunctions form the orthogonal set used in Fourier series.

---

### Explanation

To perform a Half Range Expansion, one must first identify the appropriate trigonometric basis. This basis arises naturally from solving the associated boundary-value problem:

$$
y'' + \lambda y = 0
$$

1.  **For a Sine Series:** We apply Dirichlet boundary conditions: $y(0) = 0$ and $y(L) = 0$. This forces the solution to be composed of sines.
2.  **For a Cosine Series:** We apply Neumann boundary conditions: $y'(0) = 0$ and $y'(L) = 0$. This forces the solution to be composed of cosines.

The following examples and exercises solve these BVPs to find the specific $\lambda_n$ (eigenvalues) and $y_n(x)$ (eigenfunctions/basis functions) required for these expansions.

### Solved Examples from the Book

**Example 2: Nontrivial Solutions of a BVP (Basis for Sine Series)**
**Problem:** Solve the boundary-value problem which generates the basis for a Fourier Sine Series on $(0, L)$.
$$
y'' + \lambda y = 0, \quad y(0) = 0, \quad y(L) = 0
$$
**Solution:**
We consider three cases for $\lambda$:
*   **Case I ($\lambda = 0$):** $y = c_1 x + c_2$.
    $y(0) = 0 \implies c_2 = 0$.
    $y(L) = c_1 L = 0 \implies c_1 = 0$. (Trivial solution)
*   **Case II ($\lambda < 0$):** Let $\lambda = -\alpha^2$. $y = c_1 \cosh \alpha x + c_2 \sinh \alpha x$.
    $y(0) = 0 \implies c_1 = 0$.
    $y(L) = c_2 \sinh \alpha L = 0$. Since $\sinh \alpha L \neq 0$ for $\alpha \neq 0$, $c_2 = 0$. (Trivial solution)
*   **Case III ($\lambda > 0$):** Let $\lambda = \alpha^2$. $y = c_1 \cos \alpha x + c_2 \sin \alpha x$.
    $y(0) = 0 \implies c_1 = 0$.
    The solution reduces to $y = c_2 \sin \alpha x$.
    Applying $y(L) = 0$: $c_2 \sin \alpha L = 0$.
    For a nontrivial solution ($c_2 \neq 0$), we require $\sin \alpha L = 0$.
    $$
    \alpha L = n\pi \implies \alpha = \frac{n\pi}{L}, \quad n = 1, 2, 3, \dots
    $$
    **Eigenvalues:** $\lambda_n = \alpha^2 = \frac{n^2\pi^2}{L^2}$.
    **Eigenfunctions:** $y_n(x) = \sin\left(\frac{n\pi x}{L}\right)$.

**Example 3: Checking a Specific Value**
**Problem:** Determine if the BVP $y'' + 5y = 0, y(0) = 0, y(L) = 0$ has nonzero solutions.
**Solution:**
Comparing to Example 2, $\lambda = 5$.
For a nontrivial solution, $\lambda$ must be an eigenvalue of the form $\frac{n^2\pi^2}{L^2}$.
Unless $5 = \frac{n^2\pi^2}{L^2}$ for some integer $n$, the only solution is the trivial solution $y = 0$.

### Solved Exercise Questions (from Zill 5.2)

**9.** Find eigenvalues and eigenfunctions for the BVP (Sine Series on $0 \le x \le \pi$).
$$
y'' + \lambda y = 0, \quad y(0) = 0, \quad y(\pi) = 0
$$

**Solution:**
Using the general solution for $\lambda > 0$ (Case III from Example 2):
$y = c_1 \cos \sqrt{\lambda}x + c_2 \sin \sqrt{\lambda}x$.
Apply $y(0) = 0$:
$$
c_1 \cos(0) + c_2 \sin(0) = 0 \implies c_1 = 0
$$
Apply $y(\pi) = 0$:
$$
c_2 \sin(\sqrt{\lambda}\pi) = 0
$$
For non-trivial solutions, $\sqrt{\lambda}\pi = n\pi$ for $n=1, 2, \dots$
$$
\sqrt{\lambda} = n \implies \lambda_n = n^2
$$
**Eigenfunctions:** $y_n = \sin(nx)$.

<br/>

**13.** Find eigenvalues and eigenfunctions for the BVP (Cosine Series on $0 \le x \le \pi$).
$$
y'' + \lambda y = 0, \quad y'(0) = 0, \quad y'(\pi) = 0
$$

**Solution:**
General solution for $\lambda > 0$: $y = c_1 \cos \sqrt{\lambda}x + c_2 \sin \sqrt{\lambda}x$.
Derivative: $y' = -c_1\sqrt{\lambda} \sin \sqrt{\lambda}x + c_2\sqrt{\lambda} \cos \sqrt{\lambda}x$.
Apply $y'(0) = 0$:
$$
-c_1\sqrt{\lambda}(0) + c_2\sqrt{\lambda}(1) = 0 \implies c_2 = 0
$$
The solution reduces to $y = c_1 \cos \sqrt{\lambda}x$.
Apply $y'(\pi) = 0$:
$$
y'(\pi) = -c_1\sqrt{\lambda} \sin(\sqrt{\lambda}\pi) = 0
$$
This implies $\sin(\sqrt{\lambda}\pi) = 0 \implies \sqrt{\lambda}\pi = n\pi \implies \lambda = n^2$.
Note: For $\lambda = 0$, $y = c_1x + c_2$. $y' = c_1$. $y'(0)=0 \implies c_1=0$. $y = c_2$ (constant).
This corresponds to $n=0$ in the cosine series ($a_0$ term).
**Eigenvalues:** $\lambda_n = n^2$ ($n=0, 1, 2, \dots$).
**Eigenfunctions:** $y_n = \cos(nx)$.

<br/>

**10.** Find eigenvalues and eigenfunctions.
$$
y'' + \lambda y = 0, \quad y(0) = 0, \quad y(\pi/4) = 0
$$

**Solution:**
This boundary condition generates a sine series on the interval $[0, \pi/4]$.
General solution: $y = c_1 \cos \sqrt{\lambda}x + c_2 \sin \sqrt{\lambda}x$.
$y(0) = 0 \implies c_1 = 0$.
$y(\pi/4) = c_2 \sin(\sqrt{\lambda}\frac{\pi}{4}) = 0$.
$$
\frac{\sqrt{\lambda}\pi}{4} = n\pi \implies \sqrt{\lambda} = 4n \implies \lambda = 16n^2
$$
**Eigenfunctions:** $y_n = \sin(4nx)$.

<br/>

**11.** Find eigenvalues and eigenfunctions.
$$
y'' + \lambda y = 0, \quad y'(0) = 0, \quad y(L) = 0
$$

**Solution:**
General solution: $y = c_1 \cos \sqrt{\lambda}x + c_2 \sin \sqrt{\lambda}x$.
$y'(x) = -c_1\sqrt{\lambda} \sin \sqrt{\lambda}x + c_2\sqrt{\lambda} \cos \sqrt{\lambda}x$.
Apply $y'(0) = 0$:
$$
c_2\sqrt{\lambda} = 0 \implies c_2 = 0 \quad (\text{assuming } \lambda \ne 0)
$$
Solution reduces to $y = c_1 \cos \sqrt{\lambda}x$.
Apply $y(L) = 0$:
$$
c_1 \cos(\sqrt{\lambda}L) = 0
$$
Cosine is zero at odd multiples of $\pi/2$:
$$
\sqrt{\lambda}L = \frac{(2n-1)\pi}{2} \implies \sqrt{\lambda} = \frac{(2n-1)\pi}{2L}
$$
**Eigenvalues:** $\lambda_n = \frac{(2n-1)^2\pi^2}{4L^2}$.
**Eigenfunctions:** $y_n = \cos\left(\frac{(2n-1)\pi x}{2L}\right)$.

<br/>

**12.** Find eigenvalues and eigenfunctions.
$$
y'' + \lambda y = 0, \quad y(0) = 0, \quad y'(\pi/2) = 0
$$

**Solution:**
General solution: $y = c_1 \cos \sqrt{\lambda}x + c_2 \sin \sqrt{\lambda}x$.
$y(0) = 0 \implies c_1 = 0$.
$y = c_2 \sin \sqrt{\lambda}x$.
$y' = c_2\sqrt{\lambda} \cos \sqrt{\lambda}x$.
Apply $y'(\pi/2) = 0$:
$$
c_2\sqrt{\lambda} \cos\left(\sqrt{\lambda}\frac{\pi}{2}\right) = 0
$$
$$
\sqrt{\lambda}\frac{\pi}{2} = \frac{(2n-1)\pi}{2} \implies \sqrt{\lambda} = 2n-1
$$
**Eigenvalues:** $\lambda_n = (2n-1)^2$.
**Eigenfunctions:** $y_n = \sin((2n-1)x)$.

<br/>

**14.** Find eigenvalues and eigenfunctions.
$$
y'' + \lambda y = 0, \quad y(-\pi) = 0, \quad y(\pi) = 0
$$

**Solution:**
Interval length is $2\pi$.
General solution: $y = c_1 \cos \sqrt{\lambda}x + c_2 \sin \sqrt{\lambda}x$.
Apply boundary conditions:
1. $c_1 \cos(-\sqrt{\lambda}\pi) + c_2 \sin(-\sqrt{\lambda}\pi) = 0 \implies c_1 \cos(\pi\sqrt{\lambda}) - c_2 \sin(\pi\sqrt{\lambda}) = 0$
2. $c_1 \cos(\pi\sqrt{\lambda}) + c_2 \sin(\pi\sqrt{\lambda}) = 0$
Adding equations: $2c_1 \cos(\pi\sqrt{\lambda}) = 0$.
Subtracting equations: $-2c_2 \sin(\pi\sqrt{\lambda}) = 0$.
For non-trivial solutions, either $c_1 \ne 0$ (requires $\cos=0$) or $c_2 \ne 0$ (requires $\sin=0$).
Case A ($c_1 \ne 0$): $\cos(\pi\sqrt{\lambda}) = 0 \implies \pi\sqrt{\lambda} = \frac{2n-1}{2}\pi$.
Case B ($c_2 \ne 0$): $\sin(\pi\sqrt{\lambda}) = 0 \implies \pi\sqrt{\lambda} = n\pi$.
Combining these, $\sqrt{\lambda}$ can be integer or half-integer steps, often denoted $\sqrt{\lambda} = k/2$.
**Eigenfunctions:** $\cos\left(\frac{(2n-1)x}{2}\right)$ and $\sin(nx)$.

<br/>

**15.** Find eigenvalues and eigenfunctions.
$$
y'' + 2y' + (\lambda + 1)y = 0, \quad y(0) = 0, \quad y(5) = 0
$$

**Solution:**
Characteristic equation: $m^2 + 2m + (\lambda + 1) = 0$.
$$
m = \frac{-2 \pm \sqrt{4 - 4(\lambda + 1)}}{2} = -1 \pm \sqrt{1 - \lambda - 1} = -1 \pm \sqrt{-\lambda} = -1 \pm i\sqrt{\lambda}
$$
General solution (for $\lambda > 0$): $y = e^{-x}(c_1 \cos \sqrt{\lambda}x + c_2 \sin \sqrt{\lambda}x)$.
Apply $y(0) = 0$:
$$
e^0(c_1(1) + 0) = 0 \implies c_1 = 0
$$
Apply $y(5) = 0$:
$$
e^{-5}(c_2 \sin 5\sqrt{\lambda}) = 0
$$
Since $e^{-5} \ne 0$ and $c_2 \ne 0$, we need $\sin 5\sqrt{\lambda} = 0$.
$$
5\sqrt{\lambda} = n\pi \implies \sqrt{\lambda} = \frac{n\pi}{5} \implies \lambda_n = \frac{n^2\pi^2}{25}
$$
**Eigenfunctions:** $y_n = e^{-x} \sin\left(\frac{n\pi x}{5}\right)$.

<br/>

**16.** Find eigenvalues and eigenfunctions.
$$
y'' + (\lambda + 1)y = 0, \quad y'(0) = 0, \quad y'(1) = 0
$$

**Solution:**
Let $\mu = \lambda + 1$. The equation is $y'' + \mu y = 0$.
Boundary conditions on derivatives imply a cosine series.
General solution: $y = c_1 \cos \sqrt{\mu}x + c_2 \sin \sqrt{\mu}x$.
$y' = -c_1\sqrt{\mu} \sin \sqrt{\mu}x + c_2\sqrt{\mu} \cos \sqrt{\mu}x$.
$y'(0) = 0 \implies c_2 = 0$.
$y'(1) = -c_1\sqrt{\mu} \sin \sqrt{\mu} = 0$.
$$
\sqrt{\mu} = n\pi \implies \mu = n^2\pi^2
$$
Substitute back $\lambda$: $\lambda + 1 = n^2\pi^2$.
**Eigenvalues:** $\lambda_n = n^2\pi^2 - 1$.
**Eigenfunctions:** $y_n = \cos(n\pi x)$.

<br/>

**17.** Find eigenvalues and eigenfunctions for a Cauchy-Euler BVP.
$$
x^2 y'' + xy' + \lambda y = 0, \quad y(1) = 0, \quad y(e^{\pi}) = 0
$$

**Solution:**
Cauchy-Euler form $x^2 y'' + xy' + \lambda y = 0$ transforms to $u'' + \lambda u = 0$ with substitution $x = e^t$ (or assuming $y = x^m$).
Auxiliary equation for $y=x^m$: $m(m-1) + m + \lambda = 0 \implies m^2 + \lambda = 0 \implies m = \pm i\sqrt{\lambda}$.
General solution: $y = c_1 \cos(\sqrt{\lambda} \ln x) + c_2 \sin(\sqrt{\lambda} \ln x)$.
Apply $y(1) = 0$:
$$
c_1 \cos(0) + c_2 \sin(0) = 0 \implies c_1 = 0
$$
Apply $y(e^{\pi}) = 0$:
$$
c_2 \sin(\sqrt{\lambda} \ln e^{\pi}) = c_2 \sin(\sqrt{\lambda}\pi) = 0
$$
$$
\sqrt{\lambda}\pi = n\pi \implies \sqrt{\lambda} = n \implies \lambda = n^2
$$
**Eigenfunctions:** $y_n = \sin(n \ln x)$.

<br/>

**18.** Find eigenvalues and eigenfunctions.
$$
x^2 y'' + xy' + \lambda y = 0, \quad y'(e^{-1}) = 0, \quad y(1) = 0
$$

**Solution:**
General solution: $y = c_1 \cos(\sqrt{\lambda} \ln x) + c_2 \sin(\sqrt{\lambda} \ln x)$.
Derivative:
$$
y' = -c_1 \frac{\sqrt{\lambda}}{x} \sin(\sqrt{\lambda} \ln x) + c_2 \frac{\sqrt{\lambda}}{x} \cos(\sqrt{\lambda} \ln x)
$$
Apply $y(1) = 0$:
$$
c_1(1) + c_2(0) = 0 \implies c_1 = 0
$$
Now $y = c_2 \sin(\sqrt{\lambda} \ln x)$ and $y' = c_2 \frac{\sqrt{\lambda}}{x} \cos(\sqrt{\lambda} \ln x)$.
Apply $y'(e^{-1}) = 0$:
$$
c_2 \frac{\sqrt{\lambda}}{e^{-1}} \cos(\sqrt{\lambda} \ln e^{-1}) = c_2 e \sqrt{\lambda} \cos(-\sqrt{\lambda}) = 0
$$
Since $\cos(-\theta) = \cos(\theta)$, we need $\cos(\sqrt{\lambda}) = 0$.
$$
\sqrt{\lambda} = \frac{(2n-1)\pi}{2}
$$
**Eigenvalues:** $\lambda_n = \frac{(2n-1)^2\pi^2}{4}$.
**Eigenfunctions:** $y_n = \sin\left(\frac{(2n-1)\pi}{2} \ln x\right)$.
