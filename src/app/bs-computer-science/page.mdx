# BS Computer Science

Welcome to the Bachelor of Science in Computer Science program documentation.

## Program Overview

The BS Computer Science program spans 8 semesters and covers fundamental to advanced topics in computer science including:

- Programming Fundamentals
- Data Structures and Algorithms
- Database Systems
- Web Development
- Software Engineering
- Artificial Intelligence
- And much more...
